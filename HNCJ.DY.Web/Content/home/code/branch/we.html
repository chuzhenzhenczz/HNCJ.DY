关于党员一点通

1、何为党员一点通？

党员一点通是党员信息、党员组织关系状态、党员知识学习分享、党员对当下实时政治的理解和党员思想交流的快捷、方便综合性党员内部成长不可缺少的平台。

2、为何选择党员一点通？

“如果有人问您，您的入党时间是什么时间？”“你怎么回答？我不太清楚；大概12月份吧；推算应该是12月份吧等等”，各种不确定的回答。2015年所有党员信息核查，所有的党员信息要大量的去翻阅党员档案。每年毕业生组织专业出现各种各样的问题及不便，而处于信息时代，党员材料也应同样采用现代信息化技术保存，简繁纸质档案的繁杂，节省保存时间，实现更安全的党员档案电子、纸质双备份。同时更重要的是信息查询、核对、转接以及党建知识、党员案例、视频分享等为党员学习提供更好的平台。

3、在党员一点通上可以做什么？

党员一点通可做到：1)党员电子与纸质档案双备份；2）党员信息快速核实及查询；3）党员转接更简单、快捷；4)党员知识学习更及时、丰富、直接；5）采用在党员论坛进行在线沟通交流等；6）个人党支部党内新闻的信息及时推送。
